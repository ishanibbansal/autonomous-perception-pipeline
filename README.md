# Waymo Motion Prediction Pipeline

## Objective
A perception and prediction pipeline utilizing the Waymo Open Motion Dataset to forecast multi-agent trajectories.

## Architecture
- **Compute:** Edge-simulated inference running on an on-premise Linux node (RTX 2070 Super).
- **Visualization:** Decoupled telemetry and spatial rendering via Foxglove Studio.
- **Networking:** Secure, zero-latency remote development tunnel via Tailscale.
